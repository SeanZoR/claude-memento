# Memento Examples

This directory contains example scenarios showing how Memento extracts actionable memories from coding sessions.

## Example Categories

### Development Workflow
- [Package Manager Correction](package-manager-correction.md) - When Claude uses wrong package manager
- [Test Location Convention](test-location-convention.md) - Discovering project test structure
- [Git Workflow Preference](git-workflow-preference.md) - User's git commit preferences

### Project Conventions
- [API Structure Discovery](api-structure-discovery.md) - Learning codebase organization
- [Naming Conventions](naming-conventions.md) - File and variable naming patterns

### User Preferences
- [Command Confirmation](command-confirmation.md) - Always show commands before running
- [Code Style Preferences](code-style-preferences.md) - Formatting and style choices

## How to Read Examples

Each example includes:
1. **Scenario** - What happened in the session
2. **Conversation Snippet** - The relevant exchange
3. **Memento Output** - What suggestion was generated
4. **Why It's Actionable** - Explanation of the value
